EduShareHub 📚

EduShareHub is a MERN stack based study resource sharing platform where students can upload, share, search, and download educational materials such as notes, PDFs, assignments, and study resources. The platform helps students collaborate and access learning materials easily within a centralized system.

🚀 Features
 User Authentication (Login & Signup)
 Upload Study Materials
 Download Resources
 Search Notes and Files
 User Dashboard
 Responsive UI Design
 Secure File Management
 Category-wise Resource Organization
